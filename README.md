# 🛒 E-Commerce Product Scraper using Selenium
## 📌 Project Overview

(scrap for dynamic webpage)
This project is a web scraping tool built with Python and Selenium that extracts product information from an e-commerce website.
The scraper collects important product details such as product name, price, rating, and number of reviews, and structures the data for further analysis.

This project demonstrates practical knowledge of web automation, data extraction, and dynamic website scraping.

🚀 Features

Extracts Product Name

Extracts Product Price

Extracts Product Rating

Extracts Number of Reviews

Handles dynamic web pages using Selenium

Stores extracted data in a structured format

🛠️ Technologies Used

Python

Selenium

WebDriver (ChromeDriver / GeckoDriver)

HTML Parsing

## ⚙️ Installation
1️⃣ Clone the repository
git clone https://github.com/your-username/product-scraper.git
cd product-scraper
2️⃣ Install dependencies
pip install -r requirements.txt
3️⃣ Install Selenium
pip install selenium
4️⃣ Download WebDriver

Download the appropriate WebDriver for your browser.

Example:

Chrome → ChromeDriver

## ▶️ Usage

Run the scraping script:

python scraper.py

The script will automatically:

Open the target website

Scrape product details

Extract product information

Store the collected data
